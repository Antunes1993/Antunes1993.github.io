About

This project aims to offer a dynamic tool to monitor the performance of bodybuilding exercises, indicating situations where the exercises are performed incorrectly, generating the risk of injuries.

Each user can access a complete report of their performance and be aware of which points they need to improve in their technique.

At the same time, gym administrators can do a complete mapping of the performance of their clients, so that they can act assertively at places and times with the highest peak of exercises being performed inappropriately.

Technical Information

This project is a web-based solution built with the Django framework.

In the Backend, the business rule and cardinality relationships were modeled using the django model layer and the MySQL database.

Using Django's views and urls layer , the platform's functionalities were built in a way that the user can verify his own personal data, his series of registered exercises and his result after each training.

The visualization of the pages was built using HTML5/CSS , together with the SaSS S framework.

Details in the rendering of pages were inserted in order to facilitate scalability and future maintenance of the solution as partials and predefined templates of html files.

For exercise monitoring, machine learning techniques were used to detect and track body joints together with image pre-processing techniques such as histogram equalization using the OpenCV library.
