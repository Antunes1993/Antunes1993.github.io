About

Using computer vision techniques, This project detects right and left hand joints, making this information available in real time, in order to recognize gestures and allow the user to interact with other devices only using their hands.

This project allows the user to have an immersive experience in virtual and augmented reality environments.

Technical Information

This project uses Google's artificial intelligence Media Pipe to detect the position of human hands and their joints in an image.

From the detected hand joint positions, the joints of interest are obtained and illustrated for the user using the OpenCV library . From this, the distance between the joints of interest is calculated and the value of this distance is used as the activation or deactivation threshold of devices. Through hand joint detection, it is possible to use joint positioning information to perform tasks, such as displaying or hiding images, trigger external devices and interact with elements in a virtual or augmented reality environment.
