About

This project focuses on predicting stock prices by integrating sentiment analysis with advanced machine learning models. By combining sentiment data with historical stock prices, the models aim to provide more accurate predictions and insights.

Sentiment analysis helps to gauge the market sentiment which can be crucial in understanding price movements. The integration of this sentiment data with machine learning models allows for a comprehensive approach to forecasting stock prices.

Technical Information

This project employs advanced neural network architectures, specifically CNN-biLSTM and CNN-biGRU. These models leverage Convolutional Neural Networks (CNN) to extract feature representations from sequence data, which are then processed by Bidirectional Long Short-Term Memory (biLSTM) and Bidirectional Gated Recurrent Units (biGRU) to capture complex temporal dependencies and sentiment-driven trends in stock prices.

The implementation is carried out in Python, utilizing robust libraries such as Pandas for efficient data manipulation and preprocessing, NumPy for handling numerical computations, and Matplotlib for detailed data visualization and analysis. The sentiment analysis component integrates text data from financial news and social media to enhance prediction accuracy by providing additional context to the historical stock price data.
