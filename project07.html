About

This project provides a web platform to register different types of cooking recipes in a community format.

Different people can share their recipes and learn from recipes shared by other users.

Technical Information

This project was developed using the Django web framework. Class modeling functionalities were used through the models layer of the framework, as well as the plataform's functionalities were inserted into the views layer , which in turn are responsible for rendering web pages created in HTML5 and CSS .

User registration system was developed using Django's forms module, and administrator mode , respectively. This system counts not only with login and logout functionalities but also with user's rights management .

The database used was MySQL . The database was modeled following good programming practices, as well as views and triggers were made available in order to facilitate and increase the security of operations performed on the database.
