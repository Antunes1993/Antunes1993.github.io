About

This project consists of obtaining financial data for the IBOV index and using it to create Simple Moving Averages (SMA) and Exponential Moving Averages (EMA).

The project includes the implementation of buy and sell indicators based on these moving averages, enabling users to make informed investment decisions.

Users can utilize the provided strategies or adapt them to their specific investment objectives.

Technical Information

The Simple Moving Average (SMA) is calculated using the formula:

SMA = (P1 + P2 + ... + Pn) / n

where P is the price at each period and n is the number of periods.

The Exponential Moving Average (EMA) is calculated using the formula:

EMAt = (Pt * (2 / (n + 1))) + (EMAt-1 * (1 - (2 / (n + 1))))

The project includes the implementation of buy and sell indicators based on these moving averages, enabling users to make informed investment decisions.

This project was built in Python programming language using the Pandas library, among others.
