About

This project consists of obtaining financial data from companies listed on the S&P500 and making them available for financial analysis.

In this way, the user will be able to make investment choices based on the strategies proposed in this project or adapting such strategies to their objectives.

Technical Information

This project uses a value investing approach, that is, it proposes that shares be purchased at a value below their intrinsic value .

Value investing strategies rely on a concept called multiples .Multiples are calculated by dividing a company's stock price by some measure of the company's worth - like earning and assets.

Here are a few examples of common multiples used in value investing.

  • Price-to-earnings
  • Price-to-book-value
  • Price-to-free-cash-flow


Each of the individual multiples used by value investors has its pros and cons. One way to minimize the impact of any specific multiple is by using a composite . We'll use a composite of 5 different valuation strategies in our strategy. This project was built in Python programming language using the Pandas library, among others.
